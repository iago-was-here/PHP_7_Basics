# Banco de Dados Relacionais

## Modelo Entidade/Relacionamento
- Chave primária - única, não vazia, não se altera.

- Tipos de Join : https://www.devmedia.com.br/inner-cross-left-rigth-e-full-joins/21016
  - Inner: busca dados da intersecção entre duas tabelas;
  - Outter: busca dados que estão fora da intersecção;
  - Left: busca dados associados a primeira tabela mencionada no select;
  - Right: busca dados associados a segunda tabela mencionada no select;
